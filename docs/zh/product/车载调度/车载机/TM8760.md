### 6、硬盘

.![硬盘](/产品/车载调度/车载机/TM8760/硬盘.png =500x)

硬盘安装步骤：

```安装硬盘
将硬盘锁打开
将硬盘上的螺丝拧紧
锁上硬盘锁
```

取出硬盘步骤：

```取出硬盘
将硬盘锁打开
等待车载机硬盘图标消失【重要】
将硬盘上的螺丝拧开
拔出硬盘
```

#### **①、基本设置**

##### **a. 注册**  

:::tip 注册：设置车辆参数信息
- 车牌号是调度平台下发的，不需要设置。  
- 车载机编号是出厂的唯一编号，不允许修改。
:::

.![设置车辆信息](/产品/车载调度/车载机/TM8760/设置车辆编号.png =500x)

##### **b. 音量**

:::tip 音量：设置设备的音量大小
- 报站音量
- 喊话音量
- GSM通话音量
- IP通话音量
- TTS音量
:::

<Carousel :pictures='deviceVoicePictures' height="400px" />

##### **c. 网络**

:::tip 网络：该参数由项目经理或者运维提供
- 1、主调度服务器IP / 端口
- 2、副调度服务器IP / 端口
- 3、主视频服务器IP / 端口
- 4、副视频服务器IP / 端口
- 5、设置调度域名和端口

说明：
- 1、【视频服务器域名和端口暂时不要设置，端口设置为0，默认不启用】
- 2、【卡机服务器根据现场情况设置】
:::

<Carousel :pictures='deviceSetNetworkPictures' height="400px" />


##### **d. 信号量**

:::tip 信号量：根据现场实际情况设置信号量
- 1、倒车通道：挂倒挡时要切换到的视频画面
- 2、右前门通道：打开右前门时要切换到的视频画面
- 3、右后门通道：打开右后门时要切换到的视频画面
- 4、左前门通道：打开左前门时要切换到的视频画面
- 5、左后门通道：打开左后门时要切换到的视频画面
- 6、后门通道切换触发方式：
> ①、司机开门：司机手动触发开门时触发
②、进站开门：车载机到达进站半径自动切换，此方式要求经纬度描点时准确
③、组合方式：①和②共同起作用
- 7、后门单双门开门配置：根据现场车型设置是单侧开门还是双侧开门
- 8、左转弯通道：打左转向时要切换到的视频画面
- 9、右转弯通道：打右转向时要切换到的视频画面
:::
