## 五、调试

### 1、贴标定线

> 车身左右两侧预留2米的空地，使用卷尺或者其他物品标识车身线、距离车身0.5米的线、距离车身1米的线，另外，可以在车前2米放置一个物体（摄像头能看到），以便能检测到标定效果。 

![贴标定线.png](/产品/主动安全/主动安全设备/TM9608/贴标定线.png =203x399)&nbsp;
![贴标定线-车头2米处.png](/产品/主动安全/主动安全设备/TM9608/贴标定线-车头2米处.png =369x397)

### 2、测量车身数据

![测量车身数据.png](/产品/主动安全/主动安全设备/TM9608/测量车身数据.png )

### 3、标定

①、打开上位机软件，选择`标定`

![上位机软件.png](/产品/主动安全/主动安全设备/TM9608/上位机软件.png )

②、选择`车外右后方（BSD）摄像头`进行标定

![右后方BSD摄像机标定.png](/产品/主动安全/主动安全设备/TM9608/右后方BSD摄像机标定.png )

③、标记辅助线，拖动车身线、0.5米线、1米线、车前线和实际位置重合。

![拖动辅助线.png](/产品/主动安全/主动安全设备/TM9608/拖动辅助线.png)

![拖动辅助线-2.png](/产品/主动安全/主动安全设备/TM9608/拖动辅助线-2.png)

④、点击提交，即可完成标定。
