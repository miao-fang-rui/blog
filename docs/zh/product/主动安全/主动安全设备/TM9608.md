## 五、调试

### 1、测量车身数据

![测量车身数据.png](/产品/主动安全/主动安全设备/TM9608/测量车身数据.png )

### 2、贴标定线

> ①、标识车身线
>
> ②、距离车身0.5米的线
>
> ③、距离车身1米的线
>
> ④、在车前2米放置一个物体（摄像头能看到），以便能检测到标定效果。

![贴标定线.png](/产品/主动安全/主动安全设备/TM9608/贴标定线.png =200x392)&nbsp;
![贴标定线-车头2米处.png](/产品/主动安全/主动安全设备/TM9608/贴标定线-车头2米处.png =364x390)

### 3、标定

①、打开上位机软件，选择`标定` 。

![上位机软件.png](/产品/主动安全/主动安全设备/TM9608/上位机软件.png )

②、选择`车外右后方（BSD）摄像头`进行标定。

![右后方BSD摄像机标定.png](/产品/主动安全/主动安全设备/TM9608/右后方BSD摄像机标定.png )

③、填写车辆参数。

![测量参数.png](/产品/主动安全/主动安全设备/TM9608/测量参数.png )

④、标记辅助线，拖动车身线、0.5米线、1米线、车前线和实际位置重合。

> 注意：一定要填写：车前横线到车头的距离2米。

![车前线标定.png](/产品/主动安全/主动安全设备/TM9608/车前线标定.png )

![标定线重合.png](/产品/主动安全/主动安全设备/TM9608/标定线重合.png )

④、点击提交，即可完成标定。
